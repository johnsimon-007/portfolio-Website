# 🚀 John Simon Portfolio

A modern full-stack portfolio website built to showcase my skills, projects, resume, and technical journey as a MERN Stack Developer.

## 🌟 Features

- Responsive modern UI
- About Me section
- Skills showcase
- Project showcase
- Blog system with MongoDB
- Like functionality for blogs
- Resume download
- Contact section
- Login & Signup pages
- Mobile-friendly design

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

### Deployment
- Vercel (Frontend)
- Render (Backend)
